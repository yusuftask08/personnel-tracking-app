# Personnel Tracking App

Bu uygulama, çalışanları oylamak ve bu oylamalara göre sıralamak amacıyla oluşturulmuş bir web uygulamasıdır.

## Used technologies

Bu uygulama aşağıdaki teknolojileri kullanır:

- React
- Vite
- Redux Toolkit
- Sass
- Prop-types
